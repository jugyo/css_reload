css_reload
=========

Reload CSS per 5 seconds.

Usage
-----

Add a parameter "css_reload" to request url.

css_reload ON:

    http://localhost:3000/foo?css_reload

css_reload OFF:

    http://localhost:3000/foo

It works on development environment.

Requirements
-----

- jQuery
